## Setup

```powershell
# installing dependencies
pnpm install

# build and preview the vite project
pnpm build
pnpm preview

# build and preview with base url
pnpm build:base
pnpm preview:base

# or

# installing dependencies
npm install

# build and preview the vite project
npm run build
npm run preview

# build and preview with base url
npm run build:base
npm run preview:base
```

## Dev Setup

```powershell
# installing dependencies
pnpm install

# running vite dev server
pnpm dev

# formatting src files using prettier
pnpm format

# build and preview the vite project
pnpm build
pnpm preview

# build and run with base url
pnpm build:base
pnpm preview:base

# (or)

# installing dependencies
npm install

# running vite dev server
npm run dev

# formatting src files using prettier
npm run format

# build and preview the vite project
npm run build
npm run preview

# build and preview with base url
npm run build:base
npm run preview:base
```
